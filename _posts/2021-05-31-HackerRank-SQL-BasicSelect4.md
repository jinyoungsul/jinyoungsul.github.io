# Select By ID

Query all columns for a city in  **CITY**  with the  _ID_  `1661`.

The  **CITY**  table is described as follows:

![](https://s3.amazonaws.com/hr-challenge-images/8137/1449729804-f21d187d0f-CITY.jpg)

```sql
SELECT *
FROM CITY
WHERE ID = 1661;
```
